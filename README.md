## Email Marketing Projects

This repo contains production-ready HTML email templates built with:
- Table-based layouts
- Inline CSS
- Mobile-responsive patterns
- Cross-client compatibility (Gmail, Outlook, Apple Mail)

### Categories
- Transactional Emails
- Marketing Campaigns
- Newsletters

Each project includes:
- `index.html` – final email code
- `preview.png` – visual preview
- `notes.md` – design + technical decisions
